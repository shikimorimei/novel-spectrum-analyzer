# STEP 6: 関係性ダイナミクス

## 何が分かるか

キャラ間の心理的距離が話ごとにどう変化するか。急接近するのか、じわじわ近づくのか、一度離れてから戻るのか。

## プロンプト

```
添付した小説の全話を読んだ上で、主要キャラクターの関係性を話ごとにスコアリングしてください。

■ 関係性距離スコア（0-10）：
  0 = 完全な他人、面識なし
  2 = 顔見知り、会話なし
  3 = 表面的な会話ができる
  4 = 日常的な雑談ができる
  5 = 個人的な話題に触れ始める
  6 = 個人的な話ができる、信頼が芽生えている
  7 = 互いの弱みを知っている
  8 = 深い信頼、秘密を共有
  9 = 互いを受け入れ、支え合っている
 10 = 完全な信頼と受容

■ 評価するペア（作品に合わせて変更してください）：
- 主人公 ↔ ヒロイン
- 主人公 ↔ 親友
- ヒロイン ↔ 世界（社会全体への信頼度）
- （必要に応じて追加）

■ 各話ごとに出力：
1. 各ペアのスコア（0-10）
2. その話でスコアが変動した理由（イベント）
3. 変動パターン（急上昇/漸増/後退/安定/揺り戻し）

■ 出力形式：JSON配列
[
  {
    "episode": 1,
    "pairs": {
      "主人公_ヒロイン": 2,
      "主人公_親友": 6,
      "ヒロイン_世界": 3
    },
    "events": ["初対面", "名前を知る"],
    "pattern": "安定"
  },
  ...
]
```

## 可視化プロンプト

```
以下は小説全N話の関係性スコアJSONです。
HTMLダッシュボードで可視化してください。

■ 可視化内容：
1. 全ペアの距離推移を重ねた折れ線チャート
2. 各ペアの変動パターン分類
3. 最も変動が大きかった話のハイライト
4. ステレオ配置図（左=近い関係、右=遠い関係の音場マップ）
```

## 結果の読み方

### 変動パターンの類型

| パターン | 形 | 意味 |
|---------|------|------|
| 階段型 | ■□■■□■■■ | イベントごとに段階的に近づく。最も自然 |
| S字型 | ＿＿／￣ | 序盤停滞→中盤急接近→終盤安定。恋愛ものに多い |
| ジグザグ | ／＼／＼ | 近づいたり離れたり。葛藤が多い関係 |
| ベースライン | ——— | ほぼ変動しない。安定した関係（親友ポジション） |
| V字 | ＼＿／ | 一度離れてから回復。危機を乗り越えた証 |

### 音響アナロジー

- **主要ペアの距離** ＝ メインメロディ（一番変化が大きい線）
- **安定キャラ** ＝ ベースライン（ずっと一定のスコアで支える）
- **複数ペアの同時変動** ＝ 和音（3ペア以上が同じ話で動く＝クライマックス）
