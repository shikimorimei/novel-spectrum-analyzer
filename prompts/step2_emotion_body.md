# STEP 2: 感情・身体辞書マッチ

## 何が分かるか

各話の感情カラー（怒りの話？静寂の話？）と、感情を身体で語っているか頭で語っているかの区別。

## プロンプト

```
添付した小説テキストを話ごとに分析し、感情語と身体反応語を辞書マッチで検出してください。

■ 感情語辞書（7カテゴリ）：
喜び・温かさ: 嬉し, 楽し, 幸せ, 温か, ほっと, 笑, 微笑, にこ, 明る, 輝, ドキドキ, 好き, 大好き, 安心, 穏やか, よかった, じんわり
悲しみ・喪失: 悲し, 寂し, 切な, つら, 辛, 苦し, 涙, 泣, 虚し, 痛い, 傷, 失, 消え, 壊れ, 崩れ, 減, 削, 孤独, 一人, 凹む, 落ち込, 俯
不安・動揺: 不安, 怖, 恐, ざわ, 揺れ, 震え, 焦, 戸惑, 迷, 混乱, もやもや, 緊張, 冷た, 固ま, 淀
怒り・苛立ち: 怒, 苛立, イラ, むかつ, うんざり, ふざけ, ひどい, 最悪, 嫌
驚き: 驚, びっくり, はっと, まさか, 目を見開, 初めて
覚悟・決意: 覚悟, 決意, 決めた, 踏み出, 向き合, 認め, 信じ, 選
静寂・凪: 静か, 沈黙, 黙, 無言, じっと, ぼんやり, 遠く

■ 身体反応語辞書（6カテゴリ）：
心臓・胸: 心臓, 胸, 鼓動, ざわ, 脈
手・指: 指先, 掴, 握, 手が, 手を, 拳, 震え, 両手
目・視線: 目が, 目を, 視線, 見つめ, 瞳, 横目
頭・思考: こめかみ, 頭が, 頭の, 眩暈
呼吸・声: 息, 声が, 声を, 喉, 囁, 深呼吸
温度: 冷た, 熱, 温か, 汗

■ 各話で算出する値：
1. 感情語密度 = 感情語の総出現数 ÷ 文字数 × 1000
2. 身体反応密度 = 身体反応語の総出現数 ÷ 文字数 × 1000
3. 各カテゴリ別の出現数
4. 支配的感情（最多カテゴリ）

■ 可視化：
- 7カテゴリのヒートマップ（横軸=話、縦軸=カテゴリ）
- 感情語密度 + 身体反応密度の推移折れ線
- 各話の身体部位レーダーチャート
- HTMLダッシュボードとして出力

■ 基準値：
- 感情語密度: 15-22/1000字
- 身体反応密度: 6-10/1000字
```

## 結果の読み方

| パターン | 意味 |
|---------|------|
| 感情密度 高 + 身体密度 高 | **身体で感情を語る**タイプ。読者は主人公の身体を通して感情を体験する |
| 感情密度 高 + 身体密度 低 | **言葉で感情を語る**タイプ。心情描写や独白が中心 |
| 感情密度 低 + 身体密度 高 | **書かないことで描く**タイプ。感情語を使わず身体反応だけで感情を伝える |
| 感情密度 低 + 身体密度 低 | **会話や行動で語る**タイプ。台詞と展開で読ませる |

## 重要な概念：「書かないことで描く」

感情語が少ないのに読者が感情を感じる話＝感情語の「顕在度」が低い話。
これは作品の欠陥ではなく、高度な技法の証拠であることが多いです。
