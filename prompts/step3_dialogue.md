# STEP 3: 台詞粒度分析

## 何が分かるか

台詞が「短い掛け声」か「長い語り」か。作品がインスト寄り（地の文で語る）かボーカル寄り（台詞で語る）かが分かる。

## プロンプト

```
添付した小説テキストから「」で囲まれた台詞を全て抽出し、各話について以下を算出してください。

■ 算出する指標：
1. 台詞数（本）
2. 台詞総文字数
3. 平均台詞長（字）
4. 中央値台詞長（字）
5. 最短・最長の台詞長
6. 長さ別分布：
   - 一言（～8字）：本数と割合
   - 一文（9-20字）：本数と割合
   - 複文（21-40字）：本数と割合
   - 長台詞（40字超）：本数と割合

■ 可視化：
- 各話の全台詞を長さ順に棒グラフで並べて一覧
- 話ごとの平均台詞長の推移
- 長さ分布の積み上げ棒グラフ
- HTMLダッシュボードとして出力

■ 判定基準：
- 平均8字以下 → インスト型（台詞は合いの手、地の文が楽器）
- 平均9-20字 → バランス型
- 平均25字以上 → ボーカル型（台詞が物語を歌う）
```

## 結果の読み方

**インスト型**の特徴：
- 台詞数は多いが短い（「ひでえ」「ふーん」「……」）
- 台詞は意味よりリズムとニュアンスを担当
- 情報・感情・展開は地の文と身体描写が運ぶ
- 例：ポストロック、インストバンド的

**ボーカル型**の特徴：
- 台詞1本の中に情報・感情・状況説明が詰まっている
- 台詞だけ読んでも話の流れが分かる
- 例：シンガーソングライター的

**中央値と平均の乖離**にも注目：
- 中央値 << 平均 → 大半は短いが少数の長台詞がある（ここぞの「独白」がある）
- 中央値 ≒ 平均 → 台詞の長さが均一（一定のテンポ）
