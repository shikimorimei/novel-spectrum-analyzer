# STEP 4: 新出語ダイナミクス

## 何が分かるか

物語が「新しい領域に踏み込んだ」タイミング。世界が広がる回と、既知の語彙で深める回が区別できる。複数話が必要。

## プロンプト

```
添付した小説テキストを話ごとに分割し、各話の「新出語」を分析してください。

■ 語彙の抽出方法（MeCab不要の簡易手法）：
1. 漢字2-4文字の連続 → 熟語として抽出
2. カタカナ2文字以上の連続 → カタカナ語として抽出
3. 漢字+ひらがな1-3文字 → 複合語として抽出

■ 新出語の定義：
その話で初めて登場し、それ以前のどの話にも出現していない語

■ 各話で算出する指標：
1. 総語彙数（その話に含まれるユニーク語数）
2. 新出語数（その話で初出の語数）
3. 新出率 = 新出語数 ÷ 総語彙数 × 100（%）
4. 累積語彙数（第1話から積み上げた総ユニーク語数）
5. 新出語リスト（漢字熟語・カタカナ語・複合語に分類）

■ 可視化：
- 新出率の推移折れ線（全話）
- 累積語彙数の面チャート
- 各話の主要新出語リスト（上位10-15語）
- HTMLダッシュボードとして出力

■ 基準値（チャートに参考帯として表示）：
- 60%超 = 世界拡張中（新しい楽器がステージに上がった）
- 40-55% = 安定運用（既存の楽器で変奏を重ねている）
- 40%未満 = 語彙的ミニマリズム（最少の音で最大の表現）
```

## 結果の読み方

### 下降カーブの形

新出率は基本的に話が進むほど下がります（既出語が増えるため）。重要なのはカーブの「形」：

- **急降下** → 序盤で世界を全部見せた。中盤以降は深掘り型
- **緩やかな降下** → 毎話少しずつ世界が広がり続けている
- **途中で反発（上昇）** → 新しい場所・人物・概念が導入された瞬間

### 反発ポイントに注目

下がっていた新出率が上がる話は「物語が新しい領域に踏み込んだ」サイン。
そこで導入された新出語を見ると、何が「新しかった」かが分かります。

### 終盤の低さは美点

最終話の新出率が30%台以下 ＝ 新語をほぼ必要としない ＝ 積み上げた語彙だけで物語を閉じられる。これは「エピローグは新しい楽器を足さず、既存の全楽器で最後のコードを鳴らす」ということ。
