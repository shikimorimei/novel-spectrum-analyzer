# STEP 7: 新出語の質的評価

## 何が分かるか

新出語が「なぜ生まれたか」の質的分析。同じ新出率60%でも「世界が広がった60%」と「言い換えが多いだけの60%」を区別できる。

## 前提

STEP 4 を先に実行し、各話の新出語リストを用意しておいてください。

## プロンプト

```
以下の小説テキストとSTEP4の新出語データを基に、各話の新出語を質的に分類してください。

■ 3分類の定義：

【必然】物語が要求した語彙。これがないと場面が成立しない。
  - 新しい場所の名前（屋上、カフェ、駅前）
  - 新キャラの名前・属性
  - 新しい概念（物語のテーマに関わる語）
  - プロット上不可欠な要素

【豊穣】なくても成立するが、あることで解像度が上がる語彙。
  - 具体的な食べ物・物体のディテール
  - 感覚の精密描写（匂い、音、触感の具体語）
  - 身体の新しい部位
  - 比喩、文学的表現
  - キャラの口癖・固有の語彙

【冗長】既出概念の言い換え、偶発的な語彙。
  - 助詞接続による複合語の断片
  - 汎用的な時間・場所表現（月曜日、午前中など）
  - 一度限りの説明的語句
  - 辞書マッチの誤検出的な断片

■ 各話について出力：
1. 必然 / 豊穣 / 冗長 の語数
2. 効率 = (必然 + 豊穣) ÷ 全新出語数 × 100（%）
3. 代表的な必然語（5-10語）と、なぜ必然なのかの一言理由
4. 代表的な豊穣語（3-5語）と、何の解像度を上げたか
5. その話全体の評価コメント（2-3文）

■ 出力形式：JSON配列
[
  {
    "episode": 1,
    "essential": 180,
    "rich": 95,
    "filler": 282,
    "efficiency": 49.4,
    "essential_words": [{"word":"ゲージ","reason":"世界ルールの核"},{"word":"カフェ","reason":"新しい舞台"},...],
    "rich_words": [{"word":"睫毛","reason":"身体距離の接近"},...],
    "comment": "世界構築の初期コストが大きく冗長も多いが、核となるルール語彙が確立された。"
  },
  ...
]

（ここに全話テキスト + STEP4の新出語データを添付）
```

## 結果の読み方

### 効率のU字カーブ

理想的な推移：
- **序盤（30-40%）**：世界構築のコストで冗長が多い。自然。
- **中盤（45-55%）**：安定。必然と豊穣のバランスが取れている。
- **終盤（55-65%）**：無駄のない導入。全ての語が高密度で機能。

### 効率が低い話を見つけたら

効率40%未満が中盤以降に出たら、その話の新出語を確認。
「なくても成立する言い換え」が多い場合は推敲のヒントになる。

### 効率が高い話は「ミニマリズムの美点」

新出語が少なく、かつ効率60%超 ＝ 既出語の再組み合わせで物語を動かしている。
同じ「手」でもカレーパンを渡す手、震える手、繋ぐ手——文脈で意味が変わる。
これが語彙的ミニマリズム。
