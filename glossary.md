# 🎵 音響アナロジー用語集

小説スペクトラムアナライザーで使用する音楽・音響用語と、小説における対応関係。

## 基本用語

| 音響用語 | 小説での意味 | 具体例 |
|---------|------------|--------|
| **ラウドネス** | 感情の声の大きさ | 感情語が多い＝音量が大きい |
| **テクスチャ** | 文章の手触り・質感 | 漢字が多い＝硬質、ひらがなが多い＝柔らかい |
| **サブベース** | 身体を通して伝わる低い振動 | 「胸がざわつく」「指先が冷たい」 |
| **リバーブ** | 言葉の裏に響く残響 | 表層と深層のGap。余韻。 |
| **ステレオ幅** | 関係性の広がり | キャラが多い＝音場が広い |
| **ダイナミックレンジ** | 静かな話と激しい話の落差 | 第5話(静)→第16話(崩壊)の振れ幅 |

## 楽器・パート対応

| 音楽の要素 | 小説の要素 |
|-----------|-----------|
| **ボーカル** | 台詞（セリフ） |
| **ギター** | 地の文（描写・心情） |
| **ベース** | 安定した脇役キャラ（ずっと一定の距離で支える） |
| **ドラム** | 展開のリズム（場面転換、時間経過） |
| **ストリングス** | 感情語による直接的な感情表現 |
| **エフェクター** | 比喩、文学的修辞 |

## 奏法対応

| 奏法 | 小説での対応 | 例 |
|------|------------|-----|
| **通常のフレージング** | 既出語の再組み合わせ | 「温かかった」が対象を変えて再登場 |
| **トレモロアーム** | 新概念語の急増 | 「防衛反応」「トラウマ」の初出 |
| **ハーモニクス** | 豊穣語による解像度UP | 「シャンプー」「睫毛」の具体描写 |
| **サステイン** | 既出語の意味深化 | 20話目の「一番好き」が初出 |
| **ミュート** | 感情語を意図的に使わない | 感情密度MINの崩壊回 |
| **速弾き** | 情報量の暴力 | 第1話で世界ルールを3文で提示 |
| **フィードバック** | 反復による増幅 | 「手が温かかった」の繰り返し |

## ジャンル対応

| 音楽ジャンル | 小説のタイプ | 特徴 |
|------------|------------|------|
| **J-Rock / ポストロック** | インスト型 | 身体密度高、感情密度高、台詞短い。地の文が楽器。 |
| **J-POP** | バランス型 | 会話で進むが感情もある。メロディ（構成）がうまい。 |
| **ラジオトーク** | 会話主体型 | 身体描写ゼロ、会話率高、語彙密度高。情報がサクサク流れる。 |
| **アコースティック** | 語りかけ型 | 会話で感情を運ぶ。身体描写もある。 |
| **フォーク** | 独白型 | 主人公のモノローグで感情と情景を描く。 |
| **クラシック** | 重厚文体型 | 語彙密度が高く、一文が長い。格調がある。 |

## 楽曲構造対応

| 楽曲の構成 | 小説の構成 |
|-----------|-----------|
| **イントロ** | 第1話の冒頭。世界観の提示。 |
| **Aメロ** | 日常パート。関係性の土台を作る。 |
| **Bメロ** | 緊張の高まり。伏線の回収が始まる。 |
| **サビ** | クライマックス。感情のピーク。 |
| **間奏** | 視点変更、回想、新キャラ導入。 |
| **大サビ** | 最大の感情的クライマックス。 |
| **アウトロ** | エピローグ。余韻で閉じる。 |
| **フック** | 冒頭の掴み。第1話の開始3文。 |

## 分析用語

| 用語 | 定義 |
|------|------|
| **クリッピング** | 感情語密度が27を超えるような過負荷状態。音割れリスク。 |
| **ミニマリズム** | 新出語が少ない（率30%台以下）のに感情ピークを作れている状態。 |
| **語彙的リサイクル** | 同じ語を文脈を変えて再利用し、異なる意味を持たせること。 |
| **スペクトル署名** | 7軸の数値パターン。作品固有の「音の指紋」。 |
| **リファレンス値** | 比較の基準となる中央帯。正解ではなく座標。 |
| **効率** | 新出語のうち必然+豊穣が占める割合。推敲度の指標。 |
| **実質新出率** | 全体新出率 × 効率。冗長を除いた「本当の新しさ」。 |
